# Pomodoro-Timer-App
뽀모도로 타이머 앱입니다.

# 기능 상세
- DatePicker 를 통해 타이머 시간을 설정
- 시작 버튼을 누르면 타이머가 시작되고 일시 정지를 누르면 타이머가 일시정지
- 취소 버튼을 누르면 타이머가 종료
- 카운트 다운이 완료되면 알람이 울림

# 활용 기술
- DispatchSourceTimer
- UIView Animation
